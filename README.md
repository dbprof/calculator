# CALCULATOR
Примитивный калькулятор на Arduino.
1. В наличии для прототипирования:
* 1 х клавиатура 4x4 Membrane Switch Keypad
* 1 х ЖК-дисплей 1602 + 1602 LCD конвертор в IIC/I2C
* 1 х Arduino Uno
